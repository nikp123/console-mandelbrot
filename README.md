# console-mandelbrot
kinda bored and fascinated by math in the same time so I made thiis

## Compiling
    
    gcc main.c -lm -o ok

## Running
    
    ./ok

## Controls
    
    q - to quit
    arrows do the obvious
    a/z - more detail/less detail
    s/x - zoom in/zoom out
    +/- - zoom in/zoom out
